# LC167 - Two Sum II

## Pattern

Two Pointers (Opposite Direction)

---

## Mental Model

**Search Space**

All possible pairs.

**Invariant**

The answer lies between Left and Right.

**Decision Rule**

Move the only pointer capable of improving the current sum.

**Proof**

Every eliminated pointer removes an entire set of impossible pairs.

---

## Common Mistake

Confusing loop condition with stopping rule.
