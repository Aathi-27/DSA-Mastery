# LC27 — Remove Element

## Pattern
Two Pointers — Read & Write Pointer

## Difficulty
Easy

## Confidence
4/5

---

## Problem Statement

Given an integer array `nums` and an integer `val`, remove all occurrences of `val` in-place. Return the number of elements not equal to `val`.

---

## Mental Model

Read Pointer scans.

Write Pointer stores only valid elements.

---

## Invariant

Everything before Write Pointer is finalized.

---

## Valid Element Definition

```
nums[i] != val
```

---

## Algorithm

```
write = 0
for read in range(len(nums)):
    if nums[read] != val:
        nums[write] = nums[read]
        write += 1
return write
```

---

## Common Mistake

Thinking Write Pointer moves every iteration.

**Correction:** Write Pointer moves only after writing a valid element.

---

## Relationship to Other Problems

LC283 is the same algorithm with `Valid = nums[i] != 0` instead of `nums[i] != val`.
