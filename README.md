## Learning Philosophy

```
Pattern First → Mental Model → Dry Run → Algorithm → Code → Debug
→ Same-Day Recall → +1 Day → +3 Days → +7 Days → +14 Days → +30 Days → Mixed Interview Test
```

**Core Rules:**
- Never memorize code. Memorize the *mental model*.
- Dry run before typing a single line.
- Record every personal mistake — your mistake book is your most valuable asset.
- Every session ends with a revision schedule.
- Pattern recognition is the skill. Everything else follows.

## Quick Navigation

**Start a new session:** Copy [`templates/SESSION_TEMPLATE.md`](templates/SESSION_TEMPLATE.md) → `sessions/Session_XXX.md`  
**Add a new problem:** Copy [`templates/PROBLEM_TEMPLATE.md`](templates/PROBLEM_TEMPLATE.md) → `problems/LCXXX.md`  
**Add a new pattern:** Copy [`templates/PATTERN_TEMPLATE.md`](templates/PATTERN_TEMPLATE.md) → `patterns/pattern_name.md`  
**Track mistakes:** Append to [`mistake_book/MISTAKE_BOOK.md`](mistake_book/MISTAKE_BOOK.md)
